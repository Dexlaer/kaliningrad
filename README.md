# Кёнигсберг → Калининград

Интерактивный мини-сайт об истории Калининградской области: 800 лет одной земли — от языческих пруссов и Тевтонского ордена через герцогство, королевство и Германскую империю до советского Калининграда и российского эксклава.

**Сайт:** https://dexlaer.github.io/kaliningrad/

## Что внутри

- **Карта эпох** — одна и та же территория в шести исторических срезах (~1400, 1701, 1871, 1920, 1945, сегодня): нажимаешь на год — карта перекрашивается, города меняют имена
- **«Почему эксклав»** — пошаговое объяснение в шесть шагов, от Версаля до наших дней
- **Восемь эпох** — хронология с раскрывающимися карточками, ключевыми датами и фактами
- **Семь мостов Кёнигсберга** — играбельная задача Эйлера 1736 года: попробуй пройти все мосты по одному разу (спойлер: не получится, и сайт объяснит почему)
- **Замок, которого нет** — 3D-реконструкция Королевского замка (Sketchfab, модель кафедры исторической информатики МГУ), грузится по клику
- **Квиз** — шесть вопросов для проверки усвоенного
- **Гравюрные иллюстрации** эпох и мест маршрута — сгенерированы нейросетью в едином стиле старинной гравюры

Один файл `index.html` без зависимостей и сборки — только шрифты с Google Fonts.

## Локальный запуск

Открыть `index.html` в браузере, или:

```sh
python3 -m http.server 8000
```
